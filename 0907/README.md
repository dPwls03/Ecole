### 📌 문제 설명

데이터 처리 전문가가 되고 싶은 "어피치"는 문자열을 압축하는 방법에 대해 공부를 하고 있습니다.  
최근에 대량의 데이터 처리를 위한 간단한 **비손실 압축 방법**에 대해 고민하고 있는데,  
문자열에서 **같은 값이 연속해서 나타나는 것을 그 문자의 개수와 반복되는 값으로 표현하여 더 짧은 문자열로 줄이는 방법**을 연구 중입니다.

#### 예시

- `"aabbaccc"` → `"2a2ba3c"`  
- `"abcabcdede"` → 압축되지 않음

"어피치"는 문자열을 **1개 이상의 단위로 나누어 압축하여 가장 짧은 문자열로 표현**할 수 있는 방법을 찾고자 합니다.

압축할 문자열 `s`가 주어질 때, **가장 짧은 압축 문자열의 길이**를 반환하는 함수를 작성해주세요.

#### ❌ 제한사항

- 문자열의 길이는 **1 이상 1,000 이하**입니다.
- 문자열은 **알파벳 소문자**로만 이루어져 있습니다.

---

### 💻 입출력 예

| s                                    | result |
|--------------------------------------|--------|
| `"aabbaccc"`                         | `7`    |
| `"ababcdcdababcdcd"`                 | `9`    |
| `"abcabcdede"`                       | `8`    |
| `"abcabcabcabcdededededede"`         | `14`   |
| `"xababcdcdababcdcd"`                | `17`   |

### 📖 입출력 예 설명

#### 1️⃣
- 문자열을 1개 단위로 잘라 압축했을 때, `"2a2ba3c"`가 되어 가장 짧습니다.

#### 2️⃣
- 문자열을 8개 단위로 잘라 압축했을 때, `"2ababcdcd"`가 되어 가장 짧습니다.

#### 3️⃣
- 문자열을 3개 단위로 잘라 압축했을 때, `"2abcdede"`가 되어 가장 짧습니다.

#### 4️⃣
- 여러 단위로 자를 수 있으나, 6개 단위로 자르면 `"2abcabc2dedede"`가 되어 가장 짧습니다. (길이 = 14)

#### 5️⃣
- 압축되지 않는 경우로, 문자열의 길이는 17이 됩니다.
